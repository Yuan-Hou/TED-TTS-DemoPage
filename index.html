Training-Free Intra-Utterance Control for Zero-Shot TTS

A static demo showcasing segment-level emotion transitions, duration control, and automatic prompt construction for pretrained zero-shot autoregressive TTS systems.

  • Training-free No fine-tuning or additional annotations required.
  • Intra-utterance Multi-emotion transitions and segment-level duration control.
  • Zero-shot preserved Maintains base model quality and flexibility.
This demo page is static and updated automatically from examples/data.json.

Method Overview

Segment-Aware Emotion Conditioning

Apply distinct emotional conditions to segments within a single utterance using a 2D attention mask and alignment-based segment switching for smooth transitions.

Segment-Aware Duration Steering

Modulate local pacing with duration embeddings while globally managing EOS logits to ensure consistent termination across segments.

Automatic Prompt Construction

A fine-tuned Qwen3-4B model produces structured multi-segment prompts from raw text for emotion and duration control.
